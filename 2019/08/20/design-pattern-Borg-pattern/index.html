design pattern - Borg pattern

Borg模式（Monostate）：

  • 实现单例行为，某个类的多个对象共享相同的状态。
  • 在python中，对象的属性是存在于dict中，只要在init函数中，将一个共享状态字典取代dict，就可以实现多个实例共享属性。
  • 该模式在管理数据库连接时非常有用。
